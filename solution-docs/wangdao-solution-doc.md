- 用if语句输入不同的y值


# P12★
题目描述：n个整数从小到大排列，输入一个新整数，使这N+1个整数仍然有序
#### 思路
- 先输入n的值
- 用户按升序输入n个数
- 让用户输入第n+1个数
- 将a[n]与之前的数比较，从后往前重新排序
#### 总结
排序算法
```c
    for(int i=n;i>0;i--)   //从后往前比较并排序
    {
        int temp=a[i-1];
        if(a[i-1]>a[i])
        {
            a[i-1]=a[i];
            a[i]=temp;
        }
    }
```
